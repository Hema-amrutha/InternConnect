# 🌐 InternConnect

**InternConnect** is a web-based mentorship platform that connects senior interns with students seeking internship guidance.  
It helps students learn from peers who have successfully secured internships, improving preparation efficiency and success rates.

---

## 🚀 Features

- 👩‍🎓 Student and Intern login/registration
- 🤝 Request approval workflow (students send requests, interns can accept/reject)
- 💬 Real-time chat between students and interns using **Socket.io**
- 🧾 Profile pages for interns with internship details
- 🔒 Secure authentication using **Node.js** and **Express.js**
- 💾 MongoDB database for storing users and chat history

---

## 🛠️ Tech Stack

| Component | Technology |
|------------|-------------|
| Frontend   | HTML, CSS, JavaScript |
| Backend    | Node.js, Express.js |
| Database   | MongoDB |
| Real-time Communication | Socket.io |
| Authentication | JWT / Sessions |
